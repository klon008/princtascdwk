## Терминология и структура

- [x] **Коллекция** — все карты системы; высшее достижение — собрать всю
- [x] **Серия** — страница альбома; карты не пересекаются между сериями
- [x] **Бустер** — пул карт; **Тираж** — цена, N карт, шансы (аналог баннера Genshin)
- [x] Тираж не редактируется после создания; смена правил — новый тираж
- [x] Тираж нельзя перезапустить; только копия настроек с новым id
- [x] Один активный тираж (может относиться к разным бустерам по расписанию)

## Прогресс и альбом

- [x] Альбом на сайте один, с фильтрами по сериям
- [x] Счётчик по серии и по Коллекции; без силуэтов отсутствующих карт
- [x] `!альбом` — ссылка на альбом игрока

## Экономика

- [x] Лимит открытий в день — параметр админки OBS
- [ ] **Кулдаун** между открытиями (секунды/минуты) — anti-spam, не решено
- [x] Округление компенсации за дубль — вниз (`floor`)
- [x] Pity / гарантии редкости — не в MVP

## Админка и стрим

- [~] Добавить / удалить карту игроку через админку — средний срок
- [x] Карты не влияют на другие игры бота
- [ ] **Алерт OBS** при выпадении Secret Rare — не обсуждалось
