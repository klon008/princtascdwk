# Зачарованный архив

Коллекционная галерея карт принцесс на React + Vite.

**Серия MVP «Фантастический коллекционер»:** 28 карт, 7 редкостей (Common → … → Mythic → Secret Rare), плюс **рубашка серии** (характеристика серии, не коллекционная карта).

Оригинальный макет: [Figma — Implement task from specification](https://www.figma.com/design/juAXxnNG1222lxHvAJc2Yh/Implement-task-from-specification).

Документация: [SPEC.md](./SPEC.md) · [TODO_tech.md](./TODO_tech.md) · [CHECKLIST.md](./CHECKLIST.md).

## Запуск

```powershell
npm install
npm run dev
```

Приложение откроется на http://localhost:5173/

## Сборка

```powershell
npm run build
```
